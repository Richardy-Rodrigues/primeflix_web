
# PrimeFlix

PrimeFlix é um projeto desenvolvido em React. A aplicação consiste em colocar em prática o conhecimento de APIs.

Link da aplicação: https://primeflix-web-hazel.vercel.app/

## Stack utilizada

**Front-end:** CSS

**Back-end:** Javascript, React Router
